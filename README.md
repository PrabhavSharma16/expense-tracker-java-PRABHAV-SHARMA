# 💰 Expense Tracker (Java)

## 📌 Description

This is a simple Java-based console application that helps users track their daily expenses. It allows users to add expenses, view all expenses, and calculate total spending. The data is stored in a file for future use.

---

## 🚀 Features

* Add Expense
* View All Expenses
* Show Total Spending
* Data stored using File Handling

---

## 🛠️ Technologies Used

* Java (JDK 26)
* ArrayList
* File Handling (BufferedReader, BufferedWriter)

---

## ▶️ How to Run

1. Open terminal in project folder
2. Compile files:

```
javac *.java
```

3. Run program:

```
java Main
```

---

## 📂 Project Structure

* `Main.java` → Main program (menu-driven)
* `Expense.java` → Stores expense data
* `ExpenseManager.java` → Handles logic and file operations
* `expenses.txt` → Stores saved expenses

---

## 💡 Example Usage

1. Choose option 1 to add expense
2. Enter title, amount, and category
3. View expenses or total anytime
